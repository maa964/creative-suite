# Creative Suite ロードマップ

本プロジェクトは、OSSベースのクリエイティブ・スイート統合環境を目指しています。以下の各フェーズに沿って開発を推進します。

## 現状のステータス

- **現在のバージョン**: v0.7.0
- **最新の主な追加機能**: AI Studio (Text-to-Image, Speech-to-Text, 背景除去)
- **基盤技術**: Python 3.12+, PySide6, Pillow, NumPy, huggingface-hub, (ローカルAI用: torch, diffusers等)

---

## 完了済みマイルストーン

### ✅ v0.6.0: Video/Vector エディタ基本実装

- **Photo & Paint**: レイヤー編集、ブラシエンジン、プラグイン機構の安定化
- **Video Editor**: 基本的なマルチトラック・タイムライン、動画プレビュー
- **Vector Draw**: SVGベースの基本図形描画、Undo/Redo、インポート/エクスポート

### ✅ v0.7.0: AI機能の統合 (AI Studio)

- **Text-to-Image**: Stable Diffusionとの連携（Hugging Face API / ローカル実行）
- **Speech-to-Text**: Whisperによる音声文字起こし機能
- **Background Removal**: 画像の背景自動除去機能
- **バックエンド同期**: APIキー管理、ローカル依存関係の動的チェック
