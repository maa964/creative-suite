## 開発の基本原則

- **ロードマップ優先**: 特に指示のない限り roadmap.md に従って実装を進めてください。実装後は該当フェーズに完了マークをつけ、必要に応じて新たなフェーズを追加します。

- **コードコメント**: 日本語で記述してください（技術的な慣習に反しない限り）。

## 開発・プレビュー時チェックリスト

### A. 基本要件・設計意図
- [ ] **要件充足**: 変更内容が元の要件を過不足なく満たしているか。
- [ ] **意図の明確化**: 「なぜこの変更が必要か？」が明確であり、意図がコードやコミットに反映されているか。
- [ ] **過剰設計の回避 (YAGNI)**: 将来の要件を先取りしすぎて複雑になっていないか。

### B. コード品質・保守性
- [ ] **設計原則**: SOLID, DRY などの設計原則に従っているか。
- [ ] **可読性**: 6ヶ月後の自分が読んでも理解できるか。変数の命名、関数の分割は適切か。
- [ ] **ドキュメント**: 必要なコメント、docstring、APIドキュメントが追加・更新されているか。
- [ ] **依存関係**: 不要なパッケージの追加はないか、バージョンは適切か。

### C. 安定性・セキュリティ
- [ ] **エッジケース**: `null`、空配列、境界値の処理、並行処理時の競合が考慮されているか。
- [ ] **セキュリティ**: インジェクション、XSS、認証・認可の漏れ、プライバシー保護などの懸念はないか。
- [ ] **副作用・競合**: 他のコンポーネントへの悪影響はないか、他ブランチとの競合はないか。
- [ ] **信頼性**: フェイルセーフ、リトライ、フォールバックなどのエラーハンドリングが適切か。

### D. パフォーマンス・効率
- [ ] **最適化**: N+1問題、不要な再レンダリング、メモリリークの可能性はないか。
- [ ] **効率**: レスポンスタイム、スループット、レイテンシが許容範囲内か。
- [ ] **スケーラビリティ**: 水平/垂直スケーリングや負荷分散に耐えうるか。

### E. 運用・テスト・アクセシビリティ
- [ ] **テスト**: ユニットテスト、結合テスト、あるいは手動検証が行われているか。
- [ ] **モニタリング**: 適切なロギング、メトリクス、アラートの設定が考慮されているか。
- [ ] **アクセシビリティ**: WCAG準拠、キーボード操作、スクリーンリーダーへの対応が必要か。
- [ ] **国際化 (i18n)**: 多言語化、タイムゾーン、通貨の考慮が必要か。
- [ ] **デプロイ**: CI/CDパイプライン、環境変数、設定管理に齟齬はないか。

## AI Incremental Pipeline Architecture

### Core Philosophy

各AIプロセスは純粋関数に近い振る舞いをすること: `output = f(input, config, model_version)`

- 隠れた状態に依存しない
- グローバルコンテキストを変更しない
- 暗黙の副作用に依存しない

### Incremental Execution

再実行は以下の場合**のみ**許可:
- 入力ハッシュが変更された
- 設定ハッシュが変更された
- モデルバージョンが変更された

それ以外はキャッシュされた出力を再利用すること。

### Hashing Rules

```
SHA256(serialized_input + serialized_config + model_version)
```

### DAG Specification

パイプラインは宣言的に定義。循環依存禁止、明示的な親宣言、決定論的な走査順序。

### Dirty Flag Propagation

ノードがdirtyなら下流すべてがdirty。上流はcleanのまま。サイレント無効化は禁止。

### Error Handling Policy

全ノード: 入力スキーマ検証 → タイムアウト強制 → 例外捕捉 → 構造化エラーJSON返却。オーケストレータをクラッシュさせない。

### Resource Control

GPU並行ノード制限、メモリ監視、キューバックプレッシャー、過負荷時はフェイルファスト。安定性 > スループット。

### Logging Policy

execution.log / error.log / metrics.log を分離。混合ログ禁止。全ログは機械解析可能であること。

### Guiding Principle

> Compute only what changed. Cache everything reproducible. Structure before scale.
