# Claude 開発ガイドライン

コードやファイルに変更があった場合は、必ず `README.md` および `build.ps1` を更新してください。

## プレビュー時のチェックリスト

1. **セキュリティ**: インジェクション、認証漏れ
2. **パフォーマンス**: N+1、不要な再レンダリング
3. **保守性**: 6ヶ月後の自分が読めるか
4. **エッジケース**: null、空配列、並行処理
5. **過剰設計**: YAGNI違反してないか
6. **意図の確認**: 「なぜこの変更が必要か？」が明確か。
7. **副作用の確認**: 他の部分に悪影響を与えていないか。
